# BankYar

BankYar is an intelligent, AI-first, offline-only banking SMS management application.

## 🚀 Overview

The application automatically detects incoming banking SMS messages, extracts complex financial metadata, stores them in a heavily-encrypted local database, and provides users with queryable insights, custom notes, filtering, and rich offline analytics.

**Privacy-First & Secure-by-Design:** No user information, banking logs, or financial details ever leave the device. BankYar has zero network access.

## 📐 Architecture Design

To ensure strict code quality, scalability, security, and a seamless future migration path to cross-platform (Flutter), the project follows:
- **Clean Architecture** (Separation of Presentation, Domain, and Data)
- **Feature-First Architecture** (Vertical functional slices)
- **SOLID Principles**
- **Offline & Privacy-First Guidelines**

For detailed technical specifications, low-level data structures, and the Flutter migration blueprint, read our **[Architecture Design Document (ARCHITECTURE.md)](ARCHITECTURE.md)**.
